主题自定义

2020-01-19 / 2 min read

左右宽度

在main.less中修改总体宽度，由1200px改为1300px

.main {
  max-width: 1300px;
  margin: 0 auto;
}

在sidebar.ejs修改侧边栏占比，由1/4改为1/6

<div id="sidebar" class="sidebar-wrapper lg:static lg:w-1/6">

在post.ejs修改主内容占比，由3/4改为5/6

<div class="content-wrapper py-32 lg:p-8 lg:w-5/6 post-detail animate fadeIn">

代码高亮

原主题使用的就是prism，所以去prism.js官网下载js文件和想要的主题css文件进行替换。
在main.less中搜索对应的css文件内容进行替换即可

小修改

修改完以上后，发现并不像官网样式那样。F12开发者工具对准代码区，发现了yue.css中部分代码影响了代码显示，在main.less中删除6116-6128行

侧边栏头像调整

在sidebar.ejs中操作相关内容，第7行删除 lg:mt-32以去掉顶部距离，第9行删除 lg:mt-8以去掉底部距离

侧边栏目录调宽
